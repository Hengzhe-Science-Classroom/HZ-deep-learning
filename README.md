## Features

- **Interactive visualizations** — loss landscapes, gradient flow heatmaps, attention matrices, GAN training dynamics, diffusion process animations, all on Canvas 2D / WebGL
- **KaTeX math rendering** — rigorous notation throughout
- **Exercises with hints & solutions** — from implementation tasks to theoretical derivations
- **Progress tracking** — saved locally in your browser
- **Zero dependencies** — just open the link and learn
